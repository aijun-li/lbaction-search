# lbaction-search

This is a bundle containing several launchbar searching actions for different search engines with suggestions.

Currently support:

- Baidu
- Bilibili
- Bing(国际版)
- BingCN(国内版)
- Zhihu
- Douban Movie
- NPM
- Taobao
- JD
- SMZDM

## Usage

Just clone the repository and double-click the action you need to install.
